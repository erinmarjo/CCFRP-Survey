All graphs are set up the same way - Standard ggplot grey color, larger font size etc. These are the graphs that made it into the google doc/snapshot analysis.

FONT SIZE

To change the size of the percentages above bars, change the size argument in geom_text(). To change the size of the axis title or text, change the size argument inside of the theme() function –this should be the last function on the end of each ggplot.

PLOT COLOR

-For bars that are all one color: include a fill and color argument inside of geom_col(), but OUTSIDE of the aestheic function aes(). Assign a fill and color using hexidecimal ggplot colors. Examples are ‘#000666’, #CC6600’ which have to be in quotes. You can find all the colors here: http://www.cookbook-r.com/Graphs/Colors_(ggplot2)/.

-For bars that are different colors, assign the colors manually. Add a fill and color argument INSIDE of aes() inside of geom_col(). Then add two more functions to ggplot: scale_fill_manual() and scale_color_manual() with the designated colors inside.

ORIENTATION

To turn the bars horizonally, add the function coord_flip() to ggplot
